# Electron Assets

This folder contains assets for the Electron desktop application:

- `icon.ico` - Application icon for Windows
- `icon.png` - Application icon for other platforms
- `installer.nsh` - Custom NSIS installer script (optional)

## Icon Requirements
- Windows: 256x256 ICO format
- macOS: 512x512 PNG format  
- Linux: 512x512 PNG format

## Note
Replace the placeholder icons with your actual application icons.
You can use online converters to create ICO files from PNG files.
